# ☕ White House Cafe - Amravati

Welcome to the official repository for the **White House Cafe** website. This is a modern, premium, and fully interactive web experience designed for a stylish café in Amravati.

## ✨ Key Features

- **🏠 Interactive Hero Section:** A unique "walking" animation that responds to your mouse movement, simulating an entrance into the café.
- **🍱 Order Online:** Integrated sections for quick ordering via Zomato, Swiggy, or Direct Pickup.
- **📍 Live Location:** Real-time Google Maps integration to help customers find their way.
- **📸 Aesthetic Gallery:** Stunning visuals showcasing the café's interior, exterior, and decor.
- **🍔 Modern Menu:** A clean, card-based menu grid with smooth scroll-reveal animations.
- **🎨 Premium Design:** A warm, cozy palette of Cream, Coffee Brown, and Tan with elegant typography.

## 🛠️ Technology Stack

- **HTML5:** Semantic structure for better SEO.
- **CSS3:** Custom styling with Vanilla CSS, smooth transitions, and keyframe animations.
- **JavaScript (ES6+):** Custom logic for the hero frame-sequence animation and Intersection Observer for scroll reveals.

## 📁 Project Structure

```text
WHITE HOUSE CAFE/

```

## 📝 Usage

To view the project locally:
1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Move your mouse across the hero section to control the entrance animation!
